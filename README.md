# Riba.py: a CLI tool for csv file statistics

# description: 
an app designed to automate data analysis of files with csv format

demo: (screenshot would be pasted here)

# installation: 
download .py file and launch with python3 in the terminal of choice

# usage: 
upon starting this application you'll be greeted with the introduction and asked to name a folder u need your files to be checked from. upon doing that, an app will show you the list of avaliable csv files to make  statistic calculations from. u can choose the file you need to calculate your data from the list by its assigned number. once you pick a certain number, the app will display the contents of a file to you, as well as calculate the mode, median, and a value of a middle number. PLEASE MOTE that as for right now the current version of a file can only display the data listed above when the csv file of choice is a simple list of numbers. more features will be added in the near future.

# features:
- file listing in your folder of choice
- simple statistics based on the file's contents
- a fancy fish ascii art :)

# built with:
python3

# requirements: 
you only need a base python3 and in-built modules to run this application.
